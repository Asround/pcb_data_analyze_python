# 用到的库和下载方式

```python
# 所有库:
# os(无需下载) pandas matplotlib numpy tkinter scikit-learn openpyxl

# 终端下载代码(带镜像源)
# # 基本格式: py -m pip install 模块名 -i https://pypi.tuna.tsinghua.edu.cn/simple
# # 注意一行一行复制, 不要一次性复制, 复制到终端(Terminal)而不是代码编辑页面中!
py -m pip install pandas -i https://pypi.tuna.tsinghua.edu.cn/simple
py -m pip install matplotlib -i https://pypi.tuna.tsinghua.edu.cn/simple
py -m pip install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple
py -m pip install tkinter -i https://pypi.tuna.tsinghua.edu.cn/simple
py -m pip install scikit-learn -i https://pypi.tuna.tsinghua.edu.cn/simple
py -m pip install openpyxl -i https://pypi.tuna.tsinghua.edu.cn/simple

# PyQt5 (图形化交互界面, 目前还用不到)
py -m pip install PyQt5 -i https://pypi.tuna.tsinghua.edu.cn/simple
```
